**評価指標:**

| 指標 | 定義 |
|---|---|
| 検出再現率 | 期待行のうち、重なる予測bboxが1つ以上ある行の割合 |
| 認識一致率 | 行内の予測テキストを連結し、正規化編集距離（`1 - lev/len`）で比較した平均 |
| 誤検出数 | どの期待行とも重ならない予測bboxの数 |
| 処理時間 | 3回実行の平均 / 最大 |
| OCR呼び出し回数 | Tesseract起動回数（処理時間の支配項） |
| 空振り回数 | OCRを呼んだが空文字が返った切り抜きの数 |
| 文字別 段階追跡 | 各期待文字が消えた段階（`final` までは bbox 被覆・`recognized` は最終テキストへの出現） |

## `preprocess.py` — 前処理モジュール

| 関数 | シグネチャ | 出力 | 説明 |
|---|---|---|---|
| `resize_image` | `(image, scale)` | BGR画像 | INTER_CUBIC補間で拡大 |
| `binarize_otsu` | `(gray)` | 二値画像 | Otsu法で自動しきい値二値化 |
| `binarize_adaptive` | `(gray)` | 二値画像 | 照明ムラ向け（Otsuのfallback） |
| `preprocess_for_ocr` | `(image, scale=2.0, method="otsu")` | `(binary_gray, scale)` | 拡大→グレー→二値化。倍率も返す |

**`preprocess_for_ocr` の fallback フロー:**

```
2倍 + Otsu → 信頼度低 → 4倍 + Otsu → 信頼度低 → 2倍 + Adaptive
```

## `ocr.py` — OCR処理とテキストマスク生成

### 公開API

| 関数 | シグネチャ | 出力 | 説明 |
|---|---|---|---|
| `load_image` | `(path)` | BGR画像 | 日本語パス対応（imdecode使用） |
| `run_ocr` | `(image, scale=1.0, config="--psm 11", debug=None)` | `{text_blocks, mask}` | Tesseract実行・bbox座標を元スケールに戻す |
| `run_ocr_with_preprocess` | `(image, verbose=False, debug=None)` | `{text_blocks, mask}` | MSER検出 + 切り抜きOCR。通常はこちらを使う |

**`debug` 引数（計装）:** dict を渡すと段階別スナップショットを格納する。`None`（既定）のときは何も記録せず、処理内容は計装追加前と同一。`eval_ocr.py` 専用で、本番経路には影響しない。

| キー | 内容 |
|---|---|
| `scale` | 検出スケール（`_DETECT_SCALE` = 4.0） |
| `stages` | `{段階名: [[x,y,w,h], ...]}`。**4倍スケール座標のまま**（元画像座標に丸めると段階間の同一性判定がブレるため） |
| `crops` | `[{bbox, image, raw, accepted}]`。`raw` は Tesseract の生出力で**空文字も含む**（通常経路では捨てられる情報） |
| `final` | 重複除去後の `text_blocks`（元画像座標） |
| `ocr_calls` | Tesseract呼び出し回数 |

**`text_blocks` の形式:**
```json
[
  {
    "text": "検出されたテキスト",
    "bbox": [x, y, width, height],
    "confidence": 82.0
  }
]
```

**`mask`:** 元画像と同サイズのグレースケール画像。テキスト領域が白（255）、それ以外が黒（0）。

### 内部ヘルパー

| 関数 | 説明 |
|---|---|
| `_iou(bbox1, bbox2)` | 2つのbboxのIoU計算 |
| `_deduplicate(blocks)` | 同一テキストの重複除去（信頼度高を優先）。**既知の欠陥**: Tesseractが隣接文字を含む幅広bboxを返すため、隣の別文字を重複と誤認して消す（「タ」消失の原因＝H5。段階3のCC化で廃止予定） |
| `_mser_detect_regions(gray)` | MSERで文字候補bbox検出 |
| `_filter_by_size(bboxes, ...)` | サイズフィルタ（小さすぎ・大きすぎを除外） |
| `_nms_regions(bboxes, ...)` | NMS（MSERの重複bbox除去・IoU > 0.3） |
| `_merge_horizontal(bboxes, ...)` | 横方向マージ（高さ比・縦重なり・横ギャップの3条件） |
| `_filter_by_aspect(bboxes, ...)` | アスペクト比フィルタ（細くて横長の枠線を除外） |

## パイプライン上の呼び出し順

```
load_image(path)
  ↓
run_ocr_with_preprocess(image)
  内部: MSER検出 → サイズフィルタ → NMS → 横方向マージ
        → アスペクト比フィルタ → 各領域でpreprocess_for_ocr() → run_ocr()
  ↓
result["text_blocks"]  → 空間関係推定（Phase 2）
result["mask"]         → 図形抽出（preprocess_for_shapes は Phase 2 で追加）
```
