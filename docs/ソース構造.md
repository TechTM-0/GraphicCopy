## `preprocess.py` — 前処理モジュール

| 関数 | シグネチャ | 出力 | 説明 |
|---|---|---|---|
| `resize_image` | `(image, scale)` | BGR画像 | INTER_CUBIC補間で拡大 |
| `binarize_otsu` | `(gray)` | 二値画像 | Otsu法で自動しきい値二値化 |
| `binarize_adaptive` | `(gray)` | 二値画像 | 照明ムラ向け（Otsuのfallback） |
| `preprocess_for_ocr` | `(image, scale=2.0, method="otsu")` | `(binary_gray, scale)` | 拡大→グレー→二値化。倍率も返す |

**`preprocess_for_ocr` の fallback フロー:**

```
2倍 + Otsu → 信頼度低 → 4倍 + Otsu → 信頼度低 → 2倍 + Adaptive
```

## `ocr.py` — OCR処理とテキストマスク生成

### 公開API

| 関数 | シグネチャ | 出力 | 説明 |
|---|---|---|---|
| `load_image` | `(path)` | BGR画像 | 日本語パス対応（imdecode使用） |
| `run_ocr` | `(image, scale=1.0, config="--psm 11")` | `{text_blocks, mask}` | Tesseract実行・bbox座標を元スケールに戻す |
| `run_ocr_with_preprocess` | `(image, verbose=False)` | `{text_blocks, mask}` | MSER検出 + 切り抜きOCR。通常はこちらを使う |

**`text_blocks` の形式:**
```json
[
  {
    "text": "検出されたテキスト",
    "bbox": [x, y, width, height],
    "confidence": 82.0
  }
]
```

**`mask`:** 元画像と同サイズのグレースケール画像。テキスト領域が白（255）、それ以外が黒（0）。

### 内部ヘルパー

| 関数 | 説明 |
|---|---|
| `_iou(bbox1, bbox2)` | 2つのbboxのIoU計算 |
| `_deduplicate(blocks)` | 同一テキストの重複除去（信頼度高を優先） |
| `_mser_detect_regions(gray)` | MSERで文字候補bbox検出 |
| `_filter_by_size(bboxes, ...)` | サイズフィルタ（小さすぎ・大きすぎを除外） |
| `_nms_regions(bboxes, ...)` | NMS（MSERの重複bbox除去・IoU > 0.3） |
| `_merge_horizontal(bboxes, ...)` | 横方向マージ（高さ比・縦重なり・横ギャップの3条件） |
| `_filter_by_aspect(bboxes, ...)` | アスペクト比フィルタ（細くて横長の枠線を除外） |

---

## パイプライン上の呼び出し順

```
load_image(path)
  ↓
run_ocr_with_preprocess(image)
  内部: MSER検出 → サイズフィルタ → NMS → 横方向マージ
        → アスペクト比フィルタ → 各領域でpreprocess_for_ocr() → run_ocr()
  ↓
result["text_blocks"]  → 空間関係推定（Phase 2）
result["mask"]         → 図形抽出（preprocess_for_shapes は Phase 2 で追加）
```
